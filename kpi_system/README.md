# نظام إدارة مؤشرات الأداء (KPI) للمركز الصحي الأولي

نظام ويب متكامل لإدارة ومتابعة مؤشرات الأداء (KPIs) مصمم خصيصاً للمراكز الصحية الأولية. يعمل النظام على الشبكة المحلية (LAN) ولا يتطلب اتصالاً بالإنترنت.

## 🌟 المميزات الرئيسية

- **لوحة تحكم تفاعلية**: عرض ملخص الأداء، نسبة التحقيق، والرسوم البيانية.
- **إدارة المؤشرات**: إضافة، تعديل، وحذف المؤشرات مع تحديد المستهدفات والمسؤولين.
- **إدخال البيانات**: واجهة سهلة لإدخال النتائج الشهرية والربع سنوية مع التحليل.
- **التحليل والمقارنة**: مقارنة أداء الأقسام وتحليل الاتجاهات عبر الزمن.
- **التقارير**: تصدير تقارير PDF احترافية (شهرية، ربع سنوية، سنوية، وتقارير الأقسام).
- **نظام الصلاحيات**: 4 مستويات من الصلاحيات (مدير، قسم الجودة، رئيس قسم، موظف إدخال).
- **دعم كامل للغة العربية**: واجهة مستخدم وتصدير تقارير باللغة العربية.

## 🛠️ المتطلبات التقنية

- Python 3.8 أو أحدث
- متصفح ويب حديث (Chrome, Edge, Firefox)

## 🚀 طريقة التشغيل على الشبكة المحلية (LAN)

### الخطوة 1: تثبيت المتطلبات
افتح موجه الأوامر (Command Prompt) أو الطرفية (Terminal) وقم بتشغيل الأمر التالي:
```bash
pip install flask flask-sqlalchemy flask-login pandas openpyxl fpdf2
```

### الخطوة 2: تشغيل النظام
انتقل إلى مجلد المشروع وقم بتشغيل الملف الرئيسي:
```bash
python app.py
```

### الخطوة 3: الوصول للنظام من أجهزة أخرى
1. ابحث عن عنوان IP الخاص بجهاز الخادم (الذي يعمل عليه النظام).
   - في Windows: افتح CMD واكتب `ipconfig` وابحث عن `IPv4 Address` (مثال: 192.168.1.100).
   - في Mac/Linux: افتح Terminal واكتب `ifconfig` أو `ip a`.
2. من أي جهاز آخر على نفس الشبكة، افتح المتصفح واكتب:
   ```
   http://192.168.1.100:5000
   ```
   *(استبدل 192.168.1.100 بعنوان IP الخاص بجهازك)*

| الدور | اسم المستخدم | كلمة المرور | الصلاحيات |
|-------|-------------|-------------|-----------|

## 📁 هيكل المشروع

- `app.py`: الملف الرئيسي لتشغيل التطبيق (الواجهة الخلفية).
- `models.py`: تعريف قاعدة البيانات والجداول.
- `report_generator.py`: أداة إنشاء تقارير PDF.
- `seed_data.py`: سكريبت لتحميل البيانات التجريبية.
- `kpi_database.db`: ملف قاعدة البيانات (SQLite).
- `templates/`: مجلد يحتوي على واجهات المستخدم (HTML).
- `static/`: مجلد يحتوي على ملفات التنسيق (CSS) والخطوط.
